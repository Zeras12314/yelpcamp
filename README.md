## About The Project

**YelpCamp** is a full-stack web application where users can browse, create, and review campgrounds. It features interactive maps using MapTiler, secure user authentication, and a modern UI with a “glass” aesthetic.

**Key features:**

- User authentication & authorization (signup/login/logout)
- Create, read, update, delete campgrounds
- Leave reviews for campgrounds
- Interactive maps using MapTiler
- Responsive UI with Angular components
- Backend session handling with MongoDB
